## 오늘 배운 내용
git과 gitHub를 통해 협업하는 방법. git은 개인의 기록을 갱신하면서 프로젝트를 진행할 수 있게 도와줌. gitHub는 git의 Hub라고 생각하면 편함. 동업자와 브랜치를 형성하여 push, merge를 통해(가끔은 conflict할수도있음.) 프로젝트를 절차적으로 진행할 수 있도록 도와줌. git commit, git push origin main, git add . 등의 단축어를 배웠음.
